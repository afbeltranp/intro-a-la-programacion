# Funciones y Módulos
¡Esta semana trabajaremos con Funciones y Módulos de Python!

---

## Qué Esperar de este Módulo

En este módulo repasaremos dos temas principales.
- **Funciones:**
  - Sintaxis básica de funciones.
  - Conocimiento de tipos de retorno.
  - Diferentes tipos de funciones.
  - Alcance: global, local, etc.
- **Programación con Múltiples Archivos:**
  - Módulos de Python.
  - Múltiples archivos de Python.
  - El módulo incorporado Random.

---
