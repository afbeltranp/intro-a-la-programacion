# Condicionales, Bucles y Listas

**¡Esta semana tratará sobre Condicionales, Bucles y Listas!**

---

## Qué Esperar de este Módulo

En este módulo cubriremos tres temas principales.
- **Condicionales:**
  - Cómo funcionan las declaraciones `if` y las expresiones booleanas en Python.
  - Operadores lógicos para usar con booleanos.
- **Bucles:**
  - Fundamentos de bucles `while`.
  - Fundamentos de bucles `for`.
- **Listas:**
  - Listas básicas y `range`.
  - Funciones incorporadas `min` y `max`.
  - Listas anidadas.

---

## 1. Declaraciones Condicionales

- Comprender cómo usar las declaraciones `if`, `elif` y `else` en Python para la toma de decisiones.

---

## 2. Operadores Lógicos

- Comprender cómo usar operadores lógicos (`and`, `or`, `not`) en Python.

---

## 3. Bucles `while`

- Aprender cómo implementar y controlar bucles usando **declaraciones condicionales**.

---

## 4. Listas y `range()`

- Aprender cómo usar la función `range()` para generar secuencias de números.
- Demostrar **indexación de listas** y **slicing** usando posiciones.

---

## 5. Bucles `for`

- Comprender cómo usar **bucles `for`** en Python para iterar sobre secuencias.
- Aprender cómo recorrer **listas**, **strings** y usar la función **`range()`**.

---

## 6. Min y Max en Python

- Aprender cómo usar las funciones **`min()`** y **`max()`** de Python para encontrar eficientemente los valores más pequeños y más grandes.

---

## 7. Listas Anidadas

- Comprender cómo acceder y modificar elementos en una lista 2D.
- Usar bucles anidados para iterar a través de listas 2D.

---
