# Especificaciones de Funciones de Toros y Vacas

> **Nota Importante**
En esta sección, escribe tu código en `functions.py`. Este archivo solo debe contener definiciones de funciones.


| Nombre de Función   | Tipo de Retorno | Entradas                      | Propósito                                                                                      | Detalles de Implementación                                                                                                                                                                                                                                                                                                                                                                  |
|---------------------|-----------------|-------------------------------|------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `welcome_message()` | `None`          | Ninguna                       | Muestra el mensaje de bienvenida.                                                              | Imprime: `"Welcome to Bulls and Cows!\n"`                                                                                                                                                                                                                                                                                                                                                   |
| `get_difficulty()`  | `int`           | Entrada del usuario           | Establece el nivel de dificultad (número de intentos).                                         | Aviso: <br>`"""Choose a difficulty level:\n1. Easy (12 guesses)\n2. Medium (8 guesses)\n3. Hard (5 guesses)"""` <br> Entrada: cadena de texto. <br> Si `1` → retorna `12`, si `2` → retorna `8`, si `3` → retorna `5`. <br> De lo contrario: imprime `"There was a problem processing your input. Please type 1, 2, or 3.\n"` y vuelve a solicitar hasta que sea válido.                    |
| `get_user_code()`   | `str`           | Entrada del usuario           | Permite al usuario ingresar manualmente el código secreto.                                     | Aviso: `"Type in the secret code:\n"` <br> Lee la entrada y la retorna como cadena de texto. <br> Validación: debe contener solo dígitos, tener exactamente 4 caracteres y ser todos únicos. <br> Si es inválido → imprime `"There was a problem processing your input. Please type exactly four unique digits.\n"` y vuelve a solicitar.                                                    |
| `get_secret_code()` | `str`           | Entrada del usuario (semilla) | Genera un código secreto aleatorio de 4 dígitos únicos basado en una semilla.                  | Aviso: `"Please enter a seed:\n"` <br> Convierte la semilla ingresada a entero. <br> Usa `random.seed(seed_val)`. <br> Construye un código de 4 dígitos únicos usando `random.randint(0,9)`. <br> Retorna el código como cadena de texto. <br> (Debe seguir exactamente el fragmento de implementación proporcionado.)                                                                       |
| `get_guess()`       | `str`           | Entrada del usuario           | Recopila y valida un intento.                                                                  | Aviso: `"Guess the code (type 'give up' if you would like to quit the game):\n"` <br> Si es `"give up"` → retorna inmediatamente. <br> De lo contrario, valida: debe contener solo dígitos, tener exactamente 4 caracteres y ser todos únicos. <br> Si es inválido → imprime `"There was a problem processing your input. Please type exactly four unique digits.\n"` y vuelve a solicitar. |
| `get_num_bulls()`   | `int`           | `secret` (str), `guess` (str) | Calcula el número de toros (dígito correcto en la posición correcta).                          | Compara cada índice de `secret` y `guess`. <br> Cuenta las coincidencias en la misma posición. <br> Retorna el total de toros.                                                                                                                                                                                                                                                              |
| `get_num_cows()`    | `int`           | `secret` (str), `guess` (str) | Calcula el número de vacas (dígito correcto en la posición incorrecta).                        | Por cada dígito en `guess`: si el dígito existe en `secret` pero no está en la misma posición, se cuenta como vaca. <br> Evita contar toros dos veces. <br> Retorna el total de vacas.                                                                                                                                                                                                      |
| `play_game()`       | `None`          | Entrada del usuario           | Ejecuta una ronda completa del juego.                                                          | Pasos: <br>1. Aviso: `"Type '1' if you would like the computer to generate a secret code or '2' if you want to input it yourself:\n"` → Llama a `get_secret_code()` o `get_user_code()`. <br>2. Llama a `get_difficulty()`. <br>3. Bucle del juego hasta ganar, rendirse o agotar los intentos: <br>- Llama a `get_guess()`. <br>- Si es `"give up"` → sale. <br>- Si el intento == secreto → imprime `"You win!\n"` y sale. <br>- De lo contrario: calcula toros/vacas, imprime resultados, decrementa intentos e imprime los intentos restantes. <br>4. Si se agotan los intentos: imprime `"You lose; the correct answer was XXXX"`. |
