# 🐂 Ejemplo de Toros y Vacas

## 🔗 ¡Prueba el Juego de Toros y Vacas!

Para entender mejor el juego de **Toros y Vacas** que implementarás, ¡prueba esta **aplicación web** interactiva!
Es el mismo juego de descifrar un código de 4 dígitos en el que trabajarás en tu proyecto.

- **Cómo Jugar:** 
  - Adivina el código secreto de 4 dígitos.
  - Después de cada intento, recibirás retroalimentación: 
    - **Toros** = Dígito correcto en la posición correcta.
    - **Vacas** = Dígito correcto en la posición incorrecta.

Para iniciar escribe en la siguiente terminal:
```
python demo_game.py
```
