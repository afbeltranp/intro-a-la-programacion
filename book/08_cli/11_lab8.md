## Distribución de Carga en Ingeniería Estructural

### Instrucciones

En ingeniería estructural, la **distribución de carga** es crítica al diseñar puentes, edificios y soportes mecánicos. Escribe un script de Python que:

- Acepte **dos números** como argumentos de línea de comandos:
  - El primer número representa la **carga total (N)** aplicada a una viga.
  - El segundo número representa el **número de puntos de soporte**.
- Calcule la **carga distribuida por punto de soporte** como:
  ```python
  load_per_point = total_load / num_supports
  ```
- **Maneje errores**:
  1. Si se proporciona **entrada no numérica**, imprime:
     ```plaintext
     Error: Invalid input! Enter numeric values only.
     ```
  2. Si **el número de soportes es cero**, imprime:
     ```plaintext
     Error: Cannot divide by zero! Supports must be greater than zero.
     ```

---

### Pistas

1. Usa `sys.argv` para **obtener argumentos de línea de comandos**. ¡Recuerda, `sys.argv[0]` es el nombre del script!
2. Convierte las entradas a `float` e `int` usando funciones de conversión de tipos, pero envuelve esto en un bloque `try-except` para manejar errores de conversión.
3. **Verifica si hay cero soportes** antes de realizar la división para **prevenir ZeroDivisionError**.
4. Usa cadenas formateadas (`f"{value:.2f}"`) para imprimir el **resultado redondeado a dos decimales**.
5. Asume que el usuario proporcionará el número correcto de argumentos.
